Cable Effector

The scene simulate a pressure in a cavity so solve a cable effector (you control the length of the cable.).

It uses the SurfacePressureActuator, CableEffector and QPInverseProblemSolver components, available in the SoftRobots plugin.




How to use the demo

You can change the desired length of the cable by clicking on the "CableEffector" component and changing the variable "desiredLength".

Further descriptions of the component can be found at: https://project.inria.fr/softrobot/documentation/
